Unplugged | Responsive Web Development



Overview


A fully responsive multiple page website using HTML and CSS. The project included naming, structuring and organizing site code files, modularizing the type scale with responsive typography, utilization of Flexbox Grid layouts, incorporating the HTML5 Boilerplate, responsive formatting for tablet and mobile viewports, iframes, icons, stylization, design aspects using Adobe Photoshop, adding media queries and determining responsive breakpoints, and inspecting website and testing code across browsers using developer tools and browser inspectors for responsive development.
