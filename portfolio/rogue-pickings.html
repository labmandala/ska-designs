Rogue Pickings | Responsive Web Development



Overview


An early responsive web page coded from a PSD design comp using HTML and CSS. I began the process by building out the foundation with HTML, adding content and imagery, then styling with CSS, including a CSS boilerplate. Later, the site was transformed using responsive typography, breakpoints and Flexbox to be fully responsive on mobile and tablet, in addition to the desktop version.

To responsify the site I removed floats, clears, and fixed height properties. I proceeded to center some of the content for mobile, converted the typescale, added the Flexbox styles, and then added breakpoints for tablet and desktop.
